# Data Directory

This directory contains the core data files for the library management system in CSV format.

## Contents

- **members.csv** - Library member records including membership type, status, contact information, and address
- **items.csv** - Resource catalog containing books, DVDs, and devices available for checkout
- **events.csv** - Scheduled library events and programs with descriptions and approval tracking
- **rooms.csv** - Room inventory and availability for reservations
- **transactions.csv** - Checkout and return transaction records
- **fines.csv** - Fine and fee assessment records
- **data_models.md** - Comprehensive documentation of all data entities and their fields

All data files use standard CSV format with headers and are human-readable for easy inspection and modification during testing and simulation.
